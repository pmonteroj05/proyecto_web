# Descripción de la página
Proyecto de lenguajes de marcas que consiste en la creación de una página web de temática libre. En este caso el tema general es la literatura de terror.
## Elementos originales
Contiene 5 archivos .html y un archivo .css.  

**Archivos `.html`:** 
- `index.html`: página principal o de inicio del proyecto.
- `contenido.html`: página de mayor importancia contiene dos elementos destacados que serán desarrollados en los siguientes archivos
- `detalles1.html`: primera página de detalle que ofrece mayor información sobre el elemento destacado con anterioridad (Frankenstein).
- `detalles2.html`: segunda página de detalle que ofrece mayor información sobre el elemento destacado con anterioridad (IT).
- `contacto.html`: en esta página se muestra un formulario de contacto para aquellos ususarios que quieran más información sobre el tema de la página o para informar de algún error en ella.

**Archivo `.css`:**
- `style.css`: para desarrollar los distintos estilos propios de cada una de página.

Además se han añadido recursos como imágenes en formato `.jpg` y `.svg` y dos vídeos en formato `.mp4`.
